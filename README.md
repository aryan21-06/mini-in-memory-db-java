# User Management System (In-Memory, Java)

A console-based Java application built to strengthen core backend fundamentals
without using frameworks.

## Purpose
This project was built to understand:
- Object-Oriented Programming (OOP)
- Serviceâ€“Store architecture
- Exception handling and validation flow
- Clean separation of concerns

## Features
- Add user
- Get user by ID
- Delete user
- List all users
- In-memory storage using HashMap

## Tech Stack
- Java
- No frameworks (intentional)

## Notes
This project is a foundation step before converting the same design
into a Spring Boot REST API.
